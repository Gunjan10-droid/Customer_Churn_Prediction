# 📱 Customer Churn Prediction

## 📌 Overview
Predicting whether a telecom customer will churn (leave the provider) using Machine Learning.

## 📊 Dataset
- **Source:** Kaggle — Customer Churn Prediction 2020
- **Train size:** 4250 rows, 20 columns
- **Test size:** 750 rows
- **Target:** churn (yes/no)

## 🔧 Steps Performed
1. Exploratory Data Analysis (EDA)
2. Data Preprocessing — No missing values found!
3. Label Encoding for categorical columns
4. Model Training — Random Forest Classifier
5. Prediction & Submission

## 🤖 Model Used
- Random Forest Classifier

## 📈 Results
| Metric | Score |
|--------|-------|
| Local Accuracy | 96.3% |
| Kaggle Public Score | 0.96888 |
| Kaggle Private Score | 0.94095 |
